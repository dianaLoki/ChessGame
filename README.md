# Шахматы, шашки и расширенные шахматы на Python

Консольная реализация трёх игр: классические шахматы, шашки и расширенные шахматы с нестандартными фигурами (Призрак, Змея).

## Виды игр

1. Классические шахматы — стандартные правила, рокировка, отмена ходов
2. Шашки — обычные шашки с возможностью взятия
3. Расширенные шахматы — шахматы + новые фигуры: Призрак (ходит как ладья, может перепрыгивать), Змея (ходит как король) и шашки на той же доске

## Как запустить

1. Установи Python (если ещё не установлен)
2. Скачай и запусти:
   git clone https://github.com/твой-dianaLoki/ChessGame.git
   cd ChessGame
   python main.py
3. Выбери игру: 'шахматы', 'шашки' или 'расширенные шахматы'
4. Вводи ходы в формате: e2 e4

## Возможности

- Полная реализация шахматных фигур (пешка, ладья, конь, слон, ферзь, король)
- Шашки с правилами взятия
- Дополнительные фигуры (Призрак, Змея)
- Проверка корректности ходов
- Отмена последнего хода
- Рокировка в шахматах
- Отображение доски в консоли
